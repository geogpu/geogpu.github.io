180429网页劫持

  今天网页又双叒叕（youshuangruozhuo）被hao123给劫持了，一怒之下将谷歌浏览器更名为fuckhao123，防止快捷键链接上加后缀篡改网页（然后我的webstrom测试谷歌的就跪了，破解的还没法开设置改浏览器地址）。想想当年的良心企业，又联想起百度这些年的龌龊事，就百度了下百度的历程。
  如今三大互联巨头BAT主营业务中，B的资产和业务是最差的，AI搞得不错但这波大热潮怎么发展不好评价，但起家的搜索引擎和贴吧真是日渐式微。再看看另外两家，基本上生活A娱乐T已经垄断了这两个方向，而且用户黏度极大。我不否认百度的搜索不错，贴吧我是轻度使用不做评价，网盘是我最喜欢的百度产品，他们的CDN也不错。然而，如果现在立刻开放国外互联网企业进来，可以预料绝大部分年轻人都会把搜索引擎换成谷歌。腾讯的传统qq业务会受到fcbk和tt的轻微影响，支付宝和阿里巴巴网商会受到ebay的影响。但AT不至于伤筋动骨，B的核心业务几乎一触即溃。

180503一把刻刀

  小楼今晨听风雨，独赏一曲梁间燕（斯卡布罗集市也很好听）
  最近关系不错的一个导师带了堆本科学弟学妹做毕设，把一个萌哒哒的妹子交给让我带下，抽出闲暇时间帮了一下，笨手笨脚感觉很有趣（看来我已经屌丝后期了），学妹还送我个四十多的小黑笔记本，说实话感觉还挺贵的。
  突然想起我已经快一年没怎么玩过游戏了，身为一个从小学就开始接触传奇的人，电子游戏原本已经成为了我生活的一部分（电子和书中的虚拟世界重要性不弱于我对这个真实世界的喜爱），大二下学期打lol还上瘾过一段，虽然大三就悬崖勒马233，但想想我当年第一次排位赛就十连胜黄金一也是一段传奇啊。之所以不那么疯狂了，主要这人啊，一旦有了别的目标，或者说最初的目标又有了希望还是会很快改变的。
  目标是啥呢，没错，不要问，问就是改变世界！！！
  说起来可笑，但这真不是笑话，就是改变世界。我很小时候的目标很多，回忆一下还都是无比远大的，包括但不限于每天想当国家主席还是联合国的2333）。我很幸运，父母都是中石油的员工，从小衣食无忧（还算聪明努力）公费上的也是市里最好的小学初中高中（都在一片~~所以好多好多同学跟我是相处12年+的同学与玩伴，这点我就比别人幸运得多），如今发小们有些回到家乡有些跟我一样在外漂泊学习工作，想想还有几分感慨。
  记得小时候上下学每次路过小区后面铁桥，都对桥下流过的昏黑恶臭愤恨不已，明明在我幼儿园时候它还是一条清澈的小河沟，里面还有螃蟹，怎么就变成这个样子。一颗幼小的种子就埋在了心底，后来在大概初二时候，有一次手受了伤缝针缠绷带第二天数学摸底考试用左手靠心算（光写答案没演算）考了班级数学测试第一（初中高中一直数学课代表2333）。当时简直是意气风发给我牛逼坏了，感觉等我长大简直是呼风唤雨无所不能，遂立下大誓，要推翻……（跑错片场了(lll￢ω￢)）其实就是种子发了点芽，定了个目标，一定要去改变这个世界！！首先就是把这几十公里的河道治理干净（那个时候的小目标！！！）。如今回想这点有时会觉得可笑，有时又觉得我真是志向伟大少有慧根233。
  后来化学越学越完蛋偏科严重，高三转而对计算机编程有了兴趣就把这个目标放下了，偶尔回想也不过当成一段风化的幼稚童年，随风而逝。再后来我光辉的人生戛然而止，高考没走上理想的专业，在大学里学习的热情慢慢消退，大一到大二下学期除了数学课还认真听其他都是浑浑噩噩做梦一般（现在想起就是目标缺失信仰崩塌轻微抑郁）我开始觉得世界不是那么好改变的，自己这辈子的能力可能踏不过那个门槛了。加上因为本科测绘专业身处老矿校，自己不上进又缺乏人引导，甚至当时夜不能寐怀疑自己毕业只能下黑煤窑了233也是年轻，直到中途遇见了一个年轻老师老雷同志，才在大三上学期略微重拾自信，一直到现在。
  后来快毕业时自己并不想读研（英语不好，又不爱做题，我一个靠结合专业代码毕业的人计算机二级都没过成），想着在家乡的省会找个工作就这样慢慢升职加薪平淡一生。当时自己心头常浮现的一句话就是“不折腾了，太累了”，如今想想何为哀大，哀大莫过心死。大四回家，晨跑时又路过了那座小桥，薄雾朦朦中水比原来清了许多，河面上有很多人工的净化设施（一堆象是泡沫绿色漂浮物上种了些叫不上名字的净化类植物），有小螃蟹趴在岸基边。那一瞬间，我一个一米八的工科大男生突然有点想哭，眼泪转了好几圈让我抹掉了，我以为实现不了的目标，也是有人做出来的，学环境搞这个的人们已经对这个世界做出了改变。别人的幼苗已经长成了参天大树，那我的又长成了什么鬼样子？？？以我对计算机和数学的热爱，我为什么不能在这个时代做出些什么！！！我的专业测绘中的二级学科地信不就是用来分析改变这个地球的吗！！！
  所以这也是当我有机会念研时，我没时间打游戏了，我开始加深我的c++和专业的算法，学习最新的界面库和各种算法库，趁着学生优惠买了云服务器开始向全栈发展。我已经可以写所有传统工程测绘的算法了，毕竟都是抄公式的东西，现在正在学习遥感图像和地信（web）的算法和应用程序编写。我不想去水那些论文，不想去看那些国内所谓科研界大佬的某些算法，我想研究些真正的能应用的前沿，先学到前沿再做突破！！
  天生我才必有用，终有一天我所学习的东西会成为一把刻刀，一点点改变这个世界。

180510屎一样的广告

  中午一觉起来神精气爽思路畅通地正看代码呢，突然手机一条消息打断自己，取来一看，微信群（一个小学习群）拼多多砍价的转发，真是突然间有种吃了屎的感觉，还是干噎那种。
  感觉拼多多这个平台真是恶心，欺骗傻子或者说把自己网络社交圈子看低的人进行病毒式传销（虽然看似没有什么下线交钱，但其实只不过是个变种），也许很多事情有些人说跟人平台没关系，但是把真枪发给小学生你还有理了是么。
至于我为什么把它和传销划等号，主要是这东西分析分析本质就特么是传销！！传销这种金字塔诈骗很早就有了，无非是多层级子节点病毒式供养最上层直至体系崩溃上层跑路底层倾家荡产，只不过受限于传播条件早期未能在我国显现。而其第一次大规模在我国爆发正是电子通讯（手机电话）普及的时候，通讯便捷以后，由于一个人带来的下线瞬间增加，这种模式开始有机会在我国逐渐完善的法律下慢慢变种（真是病毒一样的完美符合了进化论），每个子节点的资金上交量越来越低（可以看看美国最开始金字塔诈骗每个人上交的财富量，那种金额放中国早轻易判个几十年诈骗巨额财产了），开始加上各种包装进行伪装。
  时间就是金钱，我们常用这句话来勉励自己珍惜时间，但是实际上，又有几个人真正会将时间与资本挂钩呢。你以为那些年书信很慢，车马很远，一生只爱一个人？？？互联网时代你眼神的停留和每一次点击才是钱，才真正与货币挂钩。没仔细看腾讯在拼多多上占股多少，但就这样，一个原本依靠通讯qq起家的公司，依靠着最初的清爽的让人信赖的产品逐渐绑架用户，利用自身平台为病毒式营销（传销）提供土壤。这么一想百度还是有点良心的，再想想这是在比烂吗，只好安慰自己这只是一个小分部的无良项目组。互联网时代，依靠着人群之间的信任与信息的快速传播，一个人可以轻松把一条信息瞬间裂变百次出去。比如说我们看一次拼多多页面的价值0.1元，点进去一次价值0.2元，进去为了帮人砍价关注一下1元（实际上我很愿意花这样的价钱让人去看我的网站），百度之类的广告联盟也早有这种点网站看到广告就给钱的模式。这种拉下线依靠人际关系与信任和他人承受骚扰信息来为拼多多平台增加流量资本的模式不是传销吗。那些转发的人看不到这一点（每天忙于生计无法独立思考吗），为了蝇头小利去骚扰他人张贴网络小广告，和传销的本质又有什么区别呢。

180725革命者教义问答

  看到了一本奇书，被誉为恐怖圣经的《革命者教义问答》，作者观点很偏激并奇特，书虽诞生于反抗沙皇统治时期，但依旧被列为现代违禁数目。但不可否认，这本书的确影响了很多苏联革命时期的共产党人，当然也包括现代的恐怖分子。
  伟大的理想与偏执仅一步之隔，这是作者让人很敬佩的一点：身为一个纯粹而过于理想（偏执）的牺牲主义者，坚信要解放旧的残酷的社会，身为革命者要将全部精力放在瓦解破坏旧社会，手段可以无所不用其极。书中有一句话，“如果目标很重要，那么达到目标的手段就无关紧要”。作者的观点是，革命者就应该专心搞破坏，全心全力搞破坏，数理化都要学还要专门学搞破坏的那一部分，当推倒旧社会后立刻退位消失，把新社会的建设留给新的有伟大正常思想的年轻人。（但其所谓无关紧要的手段是极其可怕的，而且由于作者能力不低，其手段总结还很系统，就怕疯子有文化）实际上，虽然这本书在各个时期的革命者中是公认的偏激禁书，依然有很多身处高位的革命者受其影响（也可能是思想不谋而合，尤其是后期的斯大林及我国的写x某某）。道家讲阴阳平衡，腐化与偏激的革命都是不均衡的、错误的两级，我们可以在这本书中认识到一些现代恐怖分子的思想是如何的偏激（但实际上感觉如果对一个书读得不多，单纯的小白板，这类言论是很容易洗脑的）。
  有时感觉作者的言论有点类似于病变的白细疱，为了净化身体纯粹用来破坏病原微生物，但其无关紧要的手段做的确是癌细胞的事，除掉所有非革命阶级，把所有人分成不同的等级的革命者。如果每个人都有他的思想，在除掉细菌后能自己退位死亡那还好，但这就像让癌细胞可控分裂一样可笑可悲，让普通细胞接受他这种偏激白细胞的思想，那么结果就是转化为癌细胞~~专注于搞破坏的革命者并不会如同他理想那样在破除旧社会后安心退位，而是用这个充满破坏欲的大脑统治新社会，得到一个更为混乱的社会。在他这样思想下成长的年轻人也一定是为目的不择手段的年轻人，不会出现那种战争灰烬绽放渴望和平并能在和平年代长治久安的人才。
  ps：为作者的一生可叹可悲，如果每个人都有他期望的那样的觉悟，那就不必革命了，终究是个悖论。可叹其一生为理想无私奋斗，可悲在想法太简单，出发地注定通往深渊。

190218信仰（信念）

  如果没有了信仰（指信念）与理想，学习工作婚嫁无非化作一本投资书，人生一切的激情与悲伤不过是冰冷的股价折线图。在利益链条中的人不愿跳出，链条外的人想方设法钻进去，想做事的人远走他乡（不愿将就地沉浮于世），当整个习惯变成社会默契的时候，简单的正义就变成了与全社会为敌。
  几年前总不屑于国外宣传中国人没有信仰，以为他们所说的信仰就是基督耶稣mygod，赤着脸露出脖子上的青筋也要反驳自己作为儒家和道家思想传承的那份信仰。现在想想还是太年轻，宣扬神学信仰的那种是真傻子（何必与之争论），真正思考后说出这句话的牛人，其原话应该是“大部分中国人缺乏某种信念”。查了一下汉语中，信仰指对某种思想或宗教及对某人某物的信奉敬仰，注意这个“对某种思想”，所以说理解汉语还是要考虑语境，我这种纯理工的低文学素养的小年轻终归当年上了那些傻子教徒曲解原意的当啊。
  所以由此再看看国人的工作，负责产出的国企再闲适，在我这里也是正面角色。敢于创新谋求发展的无论私企国企都是神一样的存在。服务类行政类的国企闲下来那就纯垃圾，这种国企公务员等链条中基层的年轻们还偶尔读读围城矫情一下，等年纪大了顾虑多了就锈成又一代老链条。不求发展的私企那就是纯垃圾，垃圾私企依托社会关系在链条中更是污浊不堪的角色。
  当年就想学计算机，虽说被逼着选个测绘没想到这下面还有地理信息这个二级学科，Geo-IT也是爽的飞起。让我学石油和医学不如杀了我，或者说如果学了这两个我就已经行尸走肉.实际上我是对自己专业有种迷之信仰的，确切地说在理工科能安下心愿意学些本专业前沿的东西都是有些迷之自信的。
  现在社会最不缺的就是那些堆砌学历的新八股文人，这些人是学校工厂里批量诞生没有灵魂的机械蝗虫。不关心岗位的专业性，也谈不上对其喜爱与否，不需要去学习每个行业复杂的专业知识和技能，只是疯狂背考公务员、事业单位和研究生的资料，疯狂地参加他们可以参加的一切考试，干掉那些专心学习专业课沉迷某项技能的人逼着他们无事可做或同流合污。他们不关心从事的工作专业性，也谈不上对其喜爱与否，即使塞满申论和政治正确的大脑里行业的专业和小学生平齐也没有任何问题，而让其唯一胜任这个工作的原因就是其达到了社会强行规定们的学历门槛和那些废八股。我自己有点傻子一样愤愤不屑，不可否认周围人中家里有穷得确实要靠这个不谈理想搞顿饱饭解决人间疾苦，不过周围家境好的人又在干什么呢，上一辈的刻苦努力牺牲与风险造就了新一辈的得过且过？！
  ps:简单的正义变成了与全社会为敌，所以我这种想得太多的庸才才会痛苦。要是能保证回来领一份闲职位，能凑出来个环境安心搞自己喜欢这行，早就不用在这里悲天悯人了，活在安徒生的乐园中不是更好么。

190307学者商人

  产学结合，横向课题，我所看到的是，专心研究的教授天天苦读，商人一样的学者靠着吹牛一样的ppt拉投资破坏产业！
  全国第三次土地调查（三调），听着好像很nb的样子，实际上这是整个测绘行业最为劳动密集型的，科技含量最低的活。整个三调早已沦为全国各地各大测绘局和公司的内部狂欢，无数或low或屌的公司和高校老师凭着资质和关系，如同旧社会乞丐一样匍匐着高举双手恳求政府的N手外包。他们不愿负担正式员工的薪水和各种保障，用着国家的财政不需要拉动一批就业，通过可怜的薪水临时招的本科实习生们，拿着初中生培训几天都能处理的活计，美其名曰了解实际，实则浪费着学生的青春。最后用着老旧还是盗版的arcgis一顿操作，免费宣传一批美国软件，而国有地信软件平台也没有借到东风赚一分钱研发费。
  毒瘤！
  业界毒瘤！
  社会主义毒瘤！
  无论是什么公司还是什么人，都是缺德才做得出这种事。我国的地信行业的政府部门已经烂的无与伦比了，拿着吹上天的实则烂的科研都没人敢用的数据，天天硬性保密干着卖数据处理通行证的勾当。爱得多深很得多彻骨，可以查查省级政府的地信类网站，就知道这些部门对地信的作贱，依靠政权靠着卖数据和批发外包，这也算是一个国家重大基础部门该干的事？？
  我不知该怎么样评价我的老师，如今的既得利益者却又是当年学富五车之人，入学时一身正气般对我们的教育，在商业利益来临时将贪欲赤裸的爆露出来。
  我学到了理论，但现实只是让我看到了用着如何变成了新的恶龙！！！
  哀哉！哀哉！！！
  几十个本科生挤在一间小公司里天天费眼睛加班画图还不算，因为人手不够，还想借调我们这些本来就因为难得其教诲营养不良而忙于各求生路中的渣渣研究生去给他再榨几滴油，其的不负责让实验室没有统一方向，几个好学又如我这般热爱开发的人也不得各自为营，每个月都要走点弯路。
可能从这些项目中吸的血能让其另一个low的要死，专利靠改开源软件，封装api都要外包和前端都是现学的文科生的软件公司更上一层楼吧~~未来说不定还能混个省的高新技术公司呢！
  国之硕鼠！
  衣冠禽兽！
  学者商人！！

190310被淘汰的人

  我对新的技术的追逐有着狂信徒一样的迷之执着，今天看到Facebook对于空中无人机通信网的建设，查阅了好久相关资料，我突然想问一下自己什么时候有了这样的科技紧迫感。刹那间脑海浮现了那张脸——苍老、昏黄、丧失了一切——幽灵般在我的内心深处存下了一个恐怖的种子。
  大概是12年左右，我刚刚读大一，在放假和我母亲回老家的火车上。没有抢到动车和好普快车，只搞到两张便宜的老式绿皮，放下行李那一刻，我就看到了对面那双手。褶皱，层叠，好像有数不清的褶皱堆在一起，层叠间一种黑油感。
